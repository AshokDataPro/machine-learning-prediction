Key insights & improvement suggestions
✓
High R² is expected — stock prices are highly autocorrelated. close_lag1 alone can explain ~99% of variance. This is normal for next-day prediction with lagged features.
⚠
Data leakage risk — EMAs (15/25/50-day) are rolling means that include future data relative to some points. Ensure the rolling window is strictly backward-looking for each row.
⚠
No scaling applied — your code imports StandardScaler but doesn't use it. LinearRegression is scale-invariant, but scaling would help if you switch to Ridge, Lasso, or SVR.
→
Try Ridge or Lasso — your Pipeline import is unused. Ridge regression with cross-validated alpha would reduce overfitting risk and is a natural next step.
→
Add MACD or Bollinger Bands — momentum features like MACD (12/26 EMA difference) or Bollinger Band width often improve signal quality beyond RSI alone.
→
Walk-forward validation — instead of a single train/test split, rolling-window cross-validation better simulates real trading conditions and gives more robust error estimates.
